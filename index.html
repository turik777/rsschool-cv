Code Example

In this example, the fibonacci function calculates the n-th number in the Fibonacci sequence and prints it to the console.


                    function fibonacci(n) {
                        let fib = [0, 1];
                        for (i = 2; i <= n; i++) {
                            fib[i] = fib[i - 2] + fib[i - 1];
                        }
                        console.log(fib.pop());
                    }

                    fibonacci(3); // 2
                    fibonacci(7); // 13
                    fibonacci(77); // 5527939700884757
